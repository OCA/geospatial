Prerequisites¶

Python¶

geojson:

pip install geojson

Shapely:

pip install Shapely==1.2.13

For Mac user: do not forget the following directive:

ARCHFLAGS="-arch i386 -arch x86_64"

Please refer to PostGIS installation directive or to your OS package system.
